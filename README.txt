FLAG CRUSADERS WEBSITE
======================

This folder contains all the files for The Flag Crusaders website.

FILES INCLUDED:
===============
- index.html          (Main homepage)
- flag-ninjas.html    (Flag Ninjas Report page)
- donate.html         (Donation page)
- gallery.html        (Gallery page)
- contact.html        (Contact page)
- newsletter.html     (Newsletter signup page)
- members.html        (Members-only area)

HOW TO UPLOAD TO NETLIFY:
=========================
1. Go to: https://app.netlify.com/drop
2. Drag ALL 7 HTML files from this folder onto the page
   (Select all files, NOT the folder)
3. Wait 10 seconds
4. Get your URL!

IMPORTANT:
==========
- Make sure you upload the FILES, not the folder
- All files must be uploaded together
- The site will work at: https://your-site.netlify.app

TESTING:
========
After uploading, test:
- Home page loads automatically
- All navigation links work
- Mobile menu is readable
- All pages are accessible
